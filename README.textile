h1. Moving

h2. Copy database.yml

$ cp config/database.yml.postgres config/database.yml

h2. Bootstrap db

$ rake db:bootstrap

h2. Load pg db copy

$ rake db:drop
$ rake db:create
open pgsqldata file. search/replace 'pg_main' by 'your_local_pg_username'
$ psql moving_development < pgsqldata

h2. Deploy to Heroku

h4. 1. Switch to master branch

$ git checkout master

h4. 2. Compile assets

$ rake assets:precompile

h4. 3. Commit compiled assets

$ git add -A .

$ git commit -m "Compile assets"

h4. 4. Deploy to heroku

$ git push -f heroku master

h4. 5. Reset database

$ heroku pg:reset SHARED_DATABASE

h4. 6. Migrate database and run sample data on heroku

$ heroku run rake db:migrate db:seed db:sample_data

h4. 7. Restart app

$ heroku restart

h4. 8. Remove compile assets commit

$ git rebase -i HEAD~2

Follow the guideline in opened file and remove the last commit. Be careful!
